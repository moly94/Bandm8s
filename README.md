# Bandm8s

This app provides a social platform for musicians and bands to help them find other musicians for their projects.

Bandm8s (Bandmates) was created over the course of the 3rd semester at the FH Hagenberg.

Developed by:

  - Patrick Felbauer
  - Kevin Pirklbauer
